OpenArk

OpenArk is a open source toolkit on Windows. Ark is Anti Rootkit abbreviated, it aimmed at reversing/programming helper and also users can find out malwares in the OS. More and more commands will be supported in future.

  • Open Source

    One Binary, No dependences

  • Support x86/x64

    Support Windows XP/7/8/10...

  • Support Commands

    Reliable, Maintainable

Features

Process Manager
Process kill/search, enum thread/modules/window and inject...
Scanner
PE File Parser, Import/Export/Relcation Table view, RVA rebase
Bundler
Bundle files or directorys to a binary, also support script
CoderKit
View text encoding, Windows LastError/NTSTATUS
